# Titanic Data Analysis Project
## Complete End-to-End Data Science Pipeline

## Overview
A complete data science project on the Titanic
dataset demonstrating the full ML pipeline from
raw data to trained model with visualizations.

## Complete Pipeline
1. Load data (891 passengers, 12 features)
2. Missing value analysis and visualization
3. Data cleaning (3 strategies used)
4. Exploratory Data Analysis (5 analyses)
5. Feature engineering (2 new features)
6. 5 professional visualizations
7. Train 3 ML models and compare
8. Model evaluation with confusion matrix
9. Save model with pickle
10. Predict on new passengers

## Technologies Used
- Python 3, Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn (LogisticRegression,
  DecisionTree, RandomForest)
- Pickle

## Dataset
Titanic — 891 passengers (public dataset)

## Key Results
| Model               | Accuracy |
|---------------------|----------|
| Logistic Regression | ~80%     |
| Decision Tree       | ~81%     |
| Random Forest       | ~82%     |

## Key EDA Findings
- Female survival rate: 74% vs Male: 19%
- 1st class survival: 63% vs 3rd class: 24%
- Fare positively correlated with survival
- Passengers traveling alone had lower survival
